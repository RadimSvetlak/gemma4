# Přepis FLAC audio souborů pomocí gemma4 

## Postup instalace
1 Stahněte si python skripty do lokální složky na disk
2 Stáhněte si nějaké audio soubory - klidně ty FLAC z mého repo do složky ke skriptům.

## dstažení modelu
spustit soubor "Download_model.py" ten stáhne model do aktuální složky

## test modelu:
spustit příkazem "python audio_2_text.py"
